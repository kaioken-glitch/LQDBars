Version Log

March 2025 – v2.3.0
The Glass Expansion

Unified expanded player across desktop and mobile – click any cover art for a full‑screen immersive experience. Added togglable queue panel in desktop expanded view. Refined glassmorphism with enhanced backdrop blurs and gradient overlays. Fixed YouTube import duration parsing.

February 2025 – v2.2.0
Playlist Renaissance

Redesigned Playlists, Library, and Favorites with consistent glass cards. Grid/list view toggle. Detail views feature full‑screen gradient backgrounds and sticky headers. Added UpdatePopup that checks /version.json.

January 2025 – v2.1.0
YouTube Unleashed

Import entire YouTube playlists via URL – fetches video IDs and durations. Playback seamlessly switches between local audio and YouTube iframe. Debounced search and request cancellation to save API quota.

December 2024 – v2.0.0
The Great Refactor

Migrated all pages to src/pages/. PlayerContext now handles shuffle, repeat, and media session API. Drag‑to‑reorder queue and “Continue Watching” section. First release of the Liquid Bars design system.
